# spring_web_project_1
스프링부트를 이용한 회원관리 프로젝트



## 프로젝트 소개
- 스프링 프로젝트 생성
- 스프링 부트로 웹 서버 실행
- 회원 도메인 개발
- 웹 MVC 개발
- DB 연동 - JDBC, JPA, 스프링 데이터 JPA
- 테스트 케이스 작성



## 개발 기간
2023.04 ~ 2023.05



## 멤버 구성
권성준 - 회원 등록, 회원 전체 조회



## 개발 환경
- **language**: java 17
- **IDE**: IntelliJ
- **Framework**: Springboot(2.7.11)
- **Database**: H2 Database(1.4.200)



## 주요 기능
회원 등록
회원 조회
